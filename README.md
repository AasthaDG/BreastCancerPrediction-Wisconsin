# BreastCancerPrediction-Wisconsin

- Created pipeline for preprocessing and validating ML models [Logistic Regression, Ridge, Random Forest, Gradient Boosting, KNN Classifier], achieving 97% accuracy; performed feature engineering and dimensionality reduction [PCA].
- Created a FastAPI application, containerized using Docker; deployed API to AWS cloud; created real-time prediction Streamlit app; tracked experiments via MLflow on Dagshub and built JupyterBook website for project documentation.
